Naiemoji

Endpoints to use and their functionality

Endpoint Functionality
GET / Display the homepage
POST /register Registers a user
POST /login Logs in a user
GET /logout Logs out a user
GET /emojis Shows all the emojis
GET /emojis/{id} Gets a single emoji
POST /emoji Creates a new emoji
PUT /emoji/{id} Updates an emoji
PATCH /emoji/{id} Updates an emoji
DELETE /emoji/{id} Deletes an emoji
